# Projeto Site | Sistema Front End

##  Visão Geral
Este projeto representa o início da minha jornada como desenvolvedor full stack. Ele foi pensado como um sistema simples de gestão, desenvolvido inicialmente apenas no **front end**, com foco total em aprendizado, prática e organização de código.

##  Tecnologias Utilizadas
- **HTML**: responsável pela estrutura das páginas (login, menu, produtos e clientes).
- **CSS**: usado para estilização, layout responsivo, efeitos visuais e padronização da interface.
- **JavaScript**: adiciona comportamento dinâmico, como validações, modais, filtros, ordenações e controle de estado.

##  Funcionalidades Atuais
- Tela de **login fictício**, simulando autenticação.
- **Menu principal** para navegação entre as telas.
- Listagem de **produtos**, com:
  - Filtro por nome
  - Ordenação por preço
  - Modal para adicionar novos produtos
  - Seleção e exclusão de produtos
- Listagem de **clientes**, com:
  - Edição de dados via modal
  - Ativação e inativação de clientes
  - Cadastro de novos clientes
- Uso de **localStorage** para manter dados mesmo após recarregar a página.

##  Objetivo do Projeto
O objetivo não é apenas o resultado visual, mas o processo. Este projeto funciona como um ambiente de treino onde aplico conceitos reais usados no mercado, preparando a base para evoluir futuramente para:
- Separação mais clara de responsabilidades
- Código mais escalável
- Integração com back end
- APIs e banco de dados

##  Próximos Passos
- Refatorar o JavaScript em módulos
- Melhorar validações de formulário
- Criar regras de negócio mais próximas da realidade
- Implementar back end futuramente

##  Conclusão
Este projeto é meu primeiro passo prático no desenvolvimento web. Ele mostra minha evolução, minha forma de pensar soluções e minha dedicação em aprender fazendo, errando e melhorando continuamente.
