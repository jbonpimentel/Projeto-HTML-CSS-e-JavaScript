# Projeto-HTML-CSS-e-JavaScript
O projeto J-Car está sendo desenvolvido como um sistema de gerenciamento de veículos e clientes. Atualmente utiliza HTML e CSS para estrutura e layout, com telas de login, menu, produtos e clientes. O projeto está em evolução e já estou implementando JavaScript para torná-lo dinâmico, com validações, filtros e persistência de dados.
